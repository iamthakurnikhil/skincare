## Run Locally

Clone the project

```bash
  git clone https://github.com/iamthakurnikhil/skincare.git
```

Go to the project directory

```bash
  cd skincare
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
